How to cite tag {{ tag.tag }}

Go back to the tag page.

You can use one of the following citations in your LaTeX code.

  • \cite[\href{https://topological-modular-forms.github.io/the-gigantic-mess-project/tag/{{ tag.tag }}}{Tag {{ tag.tag }}}]{mess-project}
  • \cite[\href{https://topological-modular-forms.github.io/the-gigantic-mess-project/tag/{{ tag.tag }}}{% raw %}{{% endraw %}{{ tag.type | capitalize }} {{ tag.tag }}{% raw %}}{% endraw %}]{stacks-project}
  • \href{https://topological-modular-forms.github.io/the-gigantic-mess-project/tag/{{ tag.tag }}}{Tag {{ tag.tag }}}
  • \href{https://topological-modular-forms.github.io/the-gigantic-mess-project/tag/{{ tag.tag }}}{% raw %}{{% endraw %}{{ tag.type | capitalize }} {{ tag.tag }}}

You can use the following in your BibTeX file. 

@misc{stacks-project,
  author       = {The {Gigantic Mess Project Authors}},
  title        = {\textit{The Gigantic Mess Project}},
  howpublished = {\url{https://topological-modular-forms.github.io/the-gigantic-mess-project/}},
  year         = {% raw %}{{% endraw %}{{ time.year }}},
}

You can also consider using the @online type in BibLaTeX, changing the the howpublished to url. Another neat trick in BibLaTeX is the shorthand field, which you could set to MP or Mess. {% endblock %}
