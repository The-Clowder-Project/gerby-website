Welcome! This is the web version of my notes[1], including only the (mostly 100%) polished parts of them. It is built using Gerby, created by Pieter Belmans and Raymond Cheng.

Disabled Features. This website is hosted via GitHub pages, and is thus completely static, in contrast to the Stacks project, Kerodon, and the Automorphic project. As such, certain features found in those projects are disabled for this site, including the search function and comments.

Comments. Comments are most welcome however; in case you'd like to, you can contact me by email at emily.de.oliveira.santos.tmf@gmail.com .

Footnotes. There are lots of footnotes, due to how the original PDFs were created. They automatically expand when you hover your mouse over them; here's an example[2].

Placeholder Symbols. Currently, a number of macros have been defined using placeholder symbols, and look very ugly as a result. This is because I eventually plan to replace them using a custom font, which is however still WIP.

Warning: These notes are quite informal and most likely contain lots of errors, so please use them at your own risk. There are also lots of nonstandard terminology in them, so please keep this in mind as well.
