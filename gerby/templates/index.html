Welcome! This is the web version of my notes, including only the (mostly 100%) polished parts of them. It is built using Gerby, created by Pieter Belmans and Raymond Cheng.

This website is hosted via GitHub pages, and is thus completely static, in contrast to the Stacks project, Kerodon, and the Automorphic project. As such, certain features found in those projects are disabled for this site, including the search function and comments.

Comments are most welcome however; you can contact me by email at emily.de.oliveira.santos.tmf@gmail.com .

Warning: These notes are quite informal and most likely contain lots of errors, so please use them at your own risk. There are also *lots* of nonstandard terminology in them, so please keep this in mind as well.

(P.S.: The name “The Gigantic Mess Project” is a joke; I started calling my notes this because they originally started as Anki cards, and have always been a complete mess since.)
