The Clowder Project is an online reference work and wiki for category theory and ma­the­ma­ti­cs. It is built using Gerby, created by Pieter Belmans and Raymond Cheng, which is a fork of plas­TeX. The Clowder Project follows the massively collaborative model started by the Stacks Project, and aims to become the equivalent of it for ca­te­go­ry theory.

All comments, no matter how small, are ex­tre­me­ly welcome, as are contributions.

For more information about the Clowder Project, see Chapter 1: Introduction.

If you like this project, please consider supporting it on Ko-Fi:

The Clowder Project currently receives $10 monthly, which is below what is required to run the project. A larger monthly donation pool would:

  1. 1.Help ensure the continuation of the project.
  2. 2.Allow the project maintainer to devote more hours towards improving and ex­pan­ding Clowder, directly impacting how quickly new updates and content come out.
